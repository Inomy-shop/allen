# Screenshot asset directory

Future product screenshots for the documentation site should be saved here.

Each docs page lists its own suggested captures and filenames in the `:::info Media placeholders` block near the top of the page.

## Media naming convention

Docs pages include textual media placeholders that name suggested screenshot/video files. Add screenshots under this directory and videos under `static/videos/docs/` using the suggested slug names before publishing visual walkthroughs.
